      ## Introduction

      Git CMS is a headless CMS that stores content as markdown files in your GitHub repository. This post walks you through the setup process.

      ## Installation

      Add the CMS package to your Next.js app and define your block schemas in `cms.config.ts`. Mount the admin UI at any route you like.

      ## Defining blocks

      Blocks are schema-driven components. Each schema defines field names and types. The admin UI auto-generates a form from the schema, and your page renderer maps block types to React components.
